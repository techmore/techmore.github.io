Logical Build Flow

Minimal battery first

Install the control stack

GridBOSS + FlexBOSS21 gives a clean service/load integration point and automatic grid support.

Add one 16 kWh battery

The first battery buffers clouds, short spikes, solar surplus, and transition periods. It is not intended to carry winter heat overnight.

Install 9-12 kW of PV

This is the best starter size for daytime offset with one battery. Bigger PV is sensible after adding storage.

Configure self-consumption / zero export

Use CT/meter verification and then validate against Emporia mains data under sunny low-load conditions.

Expand only after data

Battery #2 is the first expansion. PV expansion toward 16-21 kW should follow once curtailment and winter behavior are known.
