Recommendation

This is a battery-value benchmark, not the default build.

The Yixiang 32 kWh pack is tempting because it offers roughly the same storage size as two EG4 16 kWh batteries for much less money. The problem is not capacity. The problem is whether the listing, BMS communication, warranty path, code acceptance, and installer comfort match the rest of the house plan.

Fit for this house

What 32 kWh means against our loads

Your measured house average is 60.5 kWh/day, and the heat pump is about 44% of mains. A 32 kWh battery is significant, but it is still not a full winter-HVAC autonomy solution.

Due diligence

Questions before using it in the EG4 plan

  • Certification: confirm actual NRTL documentation for UL1973 / UL9540 / UL9540A claims, not just marketing badges.
  • Inverter communication: verify CAN/RS485 protocol support with FlexBOSS21 before assuming closed-loop battery control.
  • Permitting: ask whether Tinicum/electrical reviewer will accept this battery in a residential ESS permit package.
  • Support: compare a 5-year overseas-vendor warranty path against EG4's ecosystem support and installer familiarity.
  • Safety: confirm disconnects, fusing, conductor sizing, grounding, labeling, and battery location requirements.

Decision

How I would use this page

Treat Yixiang as a pressure-test against EG4 battery pricing. If the installer and permit path accept it, the value is excellent. If they do not, the EG4 batteries remain the safer design choice even at the higher price.

For the current homeowner blueprint, EG4 stays the default. Yixiang becomes an alternate only after certification and compatibility are verified.

Design read

If the goal is the cheapest serious battery-backed solar system, the 12000XP deserves a separate quote. The SunGoldPower 10kW is cheaper, but the savings are only $209 against the EG4 12000XP while giving up 2kW of output, a lot of PV input headroom, and EG4 ecosystem confidence. If the goal is seamless whole-home solar/battery/grid behavior with fewer custom decisions, FlexBOSS/GridBOSS still fits better.
