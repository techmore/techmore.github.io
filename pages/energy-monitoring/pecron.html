Portable backup

Pecron E3800LFP starter plan.

A practical outage box for fridge, internet, lighting, phone charging, a standing fan, and the Jura coffee machine. Treat it as portable backup and small solar charging, not as the main HVAC or whole-home solar system.

Battery

3.84 kWh

LiFePO4 storage, about 3.2-3.5 kWh usable after AC inverter losses.

Output

4,200 W

Enough headroom for the Jura if heavy loads are managed.

Solar input

3,000 W

Two PV inputs, each 32-150V and 20A, up to 1,500W each.

Cycle rating

3,500+

Cycles to 80% capacity. Good for outages and moderate solar use; cycle count is part of the value case.

Recommendation

Buy the unit first, then mount fixed panels.

The E3800LFP is strongest as a portable battery with enough inverter headroom to run normal outage loads. For solar, use rigid mounted panels instead of foldable panels. Two large panels are the minimum useful fixed setup; four panels are the better small array if you want daytime refill to matter.

Minimum

Two 550W panels

This is the smallest mounted array worth doing. Wire one panel to each Pecron solar input, or use the final wiring your electrician prefers.

Expected refill: about 2.5-4.0 kWh on a decent sun day.

Recommended

Four 550W panels

This is the practical sweet spot if solar charging matters. Wire two panels in series per Pecron input: two strings total, one string into each input.

Expected refill: about 5-8 kWh on a decent sun day.

Not first

Six or more panels

More panels can work only if the wiring stays within voltage and current limits. At that point, this starts looking like a real permitted solar project, not a simple backup accessory.

Recommendation: stop at four unless the plan becomes permanent.

Mounting angle

A fixed small array should still be aimed by season.

The Pecron plan is still portable-first, but if we mount the 550W panels permanently the same rules apply: summer low tilt, winter steep tilt, and true south. The IR-15 / IR-30 frame is the simplest fixed choice, while the IR-45ASA is better if you actually want to tune the array through the year.

Season Recommended tilt Why
Summer15°-25°Sun is high, so flatter panels catch more light.
Winter55°-60°Sun is low, so steeper panels face it better.
Year-round fixed30°-40°A practical compromise if we do not want seasonal adjustments.
Adjustable: IR-45ASA. Fixed: IR-15 / IR-30.

Critical-loads subpanel

A small Square D panel could make this cleaner.

The useful version is not removing random circuits from the grid. It is moving a few chosen circuits into a small critical-loads subpanel, then feeding that subpanel through a code-compliant transfer setup so it can be powered by the grid normally or by the Pecron during an outage.

Good candidate circuits

Fridge

High value, moderate average load, startup surge needs inverter headroom.

Network / office

Gateway, switch, AP, NAS, and chargers if the circuit is cleanly scoped.

Selected lights

LED lighting gives a lot of comfort for very little battery draw.

Small kitchen outlet

Only if it is intentionally managed for the Jura and not shared with heavy appliances.

Component Purpose Planning note
Small Square D load center Holds the selected backup breakers. Size it for a few 120V circuits first. Do not include the 45A HVAC circuit in this Pecron plan.
Manual transfer switch or inlet/interlock Prevents grid backfeed and chooses grid or battery feed. This is the important safety part. The Pecron must never energize utility wiring accidentally.
Breaker and inlet matched to Pecron output Limits the subpanel to what the battery can safely supply. The breaker should be sized to the actual Pecron output and cable, not the old branch-circuit breaker size.
Circuit moves Moves only chosen loads from the main panel. Keep the list boring: fridge, network, a few lights, and maybe one managed outlet.

Recommendation

Yes, model a small Square D critical-loads subpanel, but keep it as an electrician-installed transfer system. It is a better fit than extension cords if you want repeatable outage behavior, and it is much cheaper and simpler than trying to make the Pecron act like the EG4 whole-home stack.

Emporia per-circuit diagnostics

Single kit versus double kit.

The Pecron is very usable with one panel kit, but two kits make the critical-loads plan far less battery-dependent and much more useful after sunrise.

Build Solar / day Critical fit Readout
1 kit / 2 panels ~2.5-4.0 kWh/day Helps extend the portable battery. Minimum mounted option.
2 kits / 4 panels ~5-8 kWh/day Much better at covering the critical-load circuit set. Recommended if solar charging matters.

Wiring note

Keep panel voltage inside the input window.

The E3800LFP solar inputs want 32-150V. The ZNShine 550W panel lists about 50.2V open-circuit voltage and about 41.9V maximum-power voltage, so two panels in series per input stays inside the 150V limit. Do not put three 550W panels in series on one Pecron input without cold-weather voltage math.
