Modern Western

A structured pathway through the Western canon, centered on Rome, Latin, history, poetry, and the value systems that shaped modern Western institutions. This is a learning lens, not an endorsement.

Pathway Classics Values History

How To Use This Path

Read in order inside each branch. If you want a single linear route, follow the recommended order below. The list is designed to teach history and values together, even when those values are disputed or uncomfortable.

Start Here

The Odyssey → The Republic → The Aeneid

Values Lens

Nicomachean Ethics → On Duties → City of God

Modern Bridge

Machiavelli → Locke → Tocqueville → Federalist

Recommended Linear Order

  1. The Odyssey → The Republic → The Aeneid
  2. Nicomachean Ethics → On Duties → The Histories (Thucydides)
  3. City of God → The Divine Comedy
  4. The Prince → Two Treatises of Government → Democracy in America
  5. The Federalist Papers → The WEIRDest People in the World
